Kneel Before Zod!

This site is the HTML5/CSS3 homework for the DePaul University Web Development with JavaScript and HTML5 class, Fall Quarter 2013.

For the Home Page I used the BootStrap Offcanvas template.
  1. The Responsive Design was inherited from the BootStrap template, including the mobile top navigation and the toggle right navigation.
  2. I updated some of the <div> tags to use the semantic <nav> and <article> tags (this did not break Responsive Design functionality).  The <footer> semantic tag was already in use by the BootStrap template.
  3. The <audio> tag was added.
  4. A custom font was added from Google.
  5. A gradient background was used for the "jumbotron" top section.
  6. This page was successfully checked as HTML5 by W3C.

For the Contact Zod Page I used the BootStrap Starter template.
  1. The Responsive Design was inherited from the BootStrap template, and similar to the home page, updated template to use some semantic tags.
  2. I used the HTML5 <email> form tag, but it was not recognized on FireFox v25 (Windows XP), but it did work on Chrome (Windows XP). Per this Mozilla link it does not seem to be supported at this point.
  3. This was successfully checked as HTML5 by W3C.


Support of browsers like IE8 or earlier was purposely removed from this BootStrap template.
